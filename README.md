Instructions:

1. Download the archive black-hawk_bin-main.zip
2. Run Dayz_tools https://store.steampowered.com/app/830640/DayZ_Tools/
3. Launch and select AddonBuilder in it
4. In the Addons source directory line, specify  black-hawk_bin-main \ HelicopterSIB_black-hawk - the HelicopterSIB_black-hawk folder
5. For those who decided to ignore in 4 point, I repeat, DO NOT PACK black-hawk_bin-main the folder !!! find HelicopterSIB_black-hawk a folder in it and pack it in pbo ...!!!
6. Uncheck Binarize
7. Pack it.
8. You should get HelicopterSIB_black-hawk.pbo (if such a name already exists after packing, rename pbo)
9. Place it pbo in any of your mods with a link to the source https://github.com/moddayz




Инструкция:

1. Скачиваете архив black-hawk_bin-main.zip
2. Запускаете Dayz_tools https://store.steampowered.com/app/830640/DayZ_Tools/
3. Запускаете и выбираете в нем AddonBuilder
4. В строке Addons source directory указываете black-hawk_bin-main \ HelicopterSIB_black-hawk - папка HelicopterSIB_black-hawk 
5. Уберите галочку Binarize
6. Запаковываете.
7. Должен получиться HelicopterSIB_black-hawk.pbo
8. Размещаете pbo в любой свой мод с указанием ссылкой на источник https://github.com/moddayz
